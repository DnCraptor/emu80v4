# Вектор-06Ц для Murmulator

Специализированная прошивка эмулятора Вектор-06Ц для RP2350/Murmulator.

Поддерживается только Pico SDK/CMake-сборка из корневого `CMakeLists.txt`.
Устаревшие desktop, Qt, SDL, wxWidgets, WebAssembly и lite-сборки удалены.

Текущее техническое задание и журнал работ находятся в
`vector06c_refactoring_plan.md`.
