# Вектор-06Ц для Murmulator

## Техническое задание и план рефакторинга

Версия: 0.6

Документ объединяет техническое задание и чек-лист выполнения работ.

## Цель

Создать специализированную прошивку только для Вектор-06Ц.

### Основные требования

-   удалить мультиэмуляторность;
-   удалить стартовую страницу;
-   убрать динамическую загрузку конфигураций;
-   максимально упростить архитектуру;
-   убрать лишнюю виртуализацию;
-   обеспечить запуск без SD-карты;
-   использовать встроенные ROM;
-   оставить один физический видеорежим (800×600).

Работы выполняются в ветке **vector06c**.

## Архитектурные решения

### Процессор

-   8080A и Z80.
-   Переключение через меню.
-   После смены --- подтверждение, сохранение и рестарт виртуальной
    машины.

### Скорость CPU

-   Несколько фиксированных частот.
-   Unlimited.
-   Меняется без рестарта.

### Видео

-   Только 800×600.

### ROM

-   Штатные ROM встроены во flash.
-   Replacement ROM загружается с SD.
-   При отсутствии replacement используется встроенный ROM с
    предупреждением.

### Носители

-   2×FDD.
-   HDD.
-   Контроллеры отключаемые.
-   Образы можно извлекать.

#### Read Only

-   Галочка RO.
-   ScrLk переключает RO.
-   Повторное открытие уже используемого образа --- только RO.
-   Запись возвращает Write Protect.

### Магнитофон

-   Отключаемый.

### Звук

-   AY-3-8910 отключаемый.
-   Covox отключаемый.

### RTC

-   Начальное время = время сборки.
-   Далее часы идут программно.
-   Не сохраняется.

### Snapshot

-   Собственный формат.
-   12 слотов.
-   Left Win+F1..F12 --- сохранить.
-   Right Win+F1..F12 --- загрузить.
-   Полный дамп состояния устройств.
-   Без сжатия.
-   Блокирующее сохранение.
-   Во время операций: пауза, volume=0, VGA продолжает вывод.

### Меню

Разделы:

-   Процессор
-   Носители
-   ROM
-   Звук
-   Магнитофон
-   Снэпшоты
-   Видео
-   Система
-   О программе

### Горячие клавиши

-   Esc --- меню.
-   Alt+Del --- Soft Reset.
-   Ctrl+Alt+Del --- Hard Reset.
-   Left Win+F1..F12 --- сохранить snapshot.
-   Right Win+F1..F12 --- загрузить snapshot.
